# 02 - Metadata & Document Settings

## Overview
This folder demonstrates the essential metadata tags and document settings in the HTML `<head>` section that control how your webpage appears and functions.

## Topics Covered

### 1. **Page Title & Charset** (`page-title-and-charset.html`)
- Learn the importance of the `<title>` tag for browser tabs and bookmarks
- Set proper character encoding with `<meta charset="UTF-8">` to display international characters correctly

### 2. **Meta Description & Keywords** (`meta-description-keywords.html`)
- Use `<meta name="description">` to provide search engine descriptions
- Understand how metadata affects SEO and search result appearance
- Learn about legacy `<meta name="keywords">` tag

### 3. **Linking CSS & JavaScript** (`linking-css-and-js.html`)
- Link external stylesheets using `<link rel="stylesheet" href="path.css">`
- Load external JavaScript files using `<script src="path.js">`
- Understand why separating concerns improves performance and maintainability

### 4. **Viewport & Responsive Design** (`viewport-and-responsive-intro.html`)
- Use `<meta name="viewport">` to make pages mobile-friendly
- Control how browsers scale content on different devices
- Learn the basics of responsive web design

## Key Takeaway
The `<head>` section contains invisible metadata and configurations that are crucial for:
- **User Experience** (title, viewport)
- **SEO** (description, charset)
- **Performance** (linking external resources)
- **Accessibility** (proper character encoding)
